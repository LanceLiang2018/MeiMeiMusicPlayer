# 简介
美美播放器，基于Androlua开发
## 美美播放器实现以下功能
* 扫描手机上的歌曲，并保存为歌曲列表
* 播放列表中的歌曲
* 显示播放进度
* 跳转到指定位置播放
* 列表歌曲循环播放
